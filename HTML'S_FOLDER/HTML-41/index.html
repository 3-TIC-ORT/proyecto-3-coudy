HTML

Div

El elemento <div> se utiliza como contenedor para otros elementos HTML. Este se utiliza a menudo para agrupar secciones de una página web o contenido, de modo que se pueda diseñar fácilmente.

Sintaxis:

<div class="Contenedor">

<h1> Esto es un encabezado en un div </h1>

<p> Esto es un párrafo en un div </p>

</div>

Así se vería la línea de código anterior:

Esto es un encabezado en un div

Esto es un párrafo en un div

PD: El color y tamaño del fondo está declarado en una hoja de estilos que veremos más adelante
