Documento HTML

La anatomía básica de un documento HTML es algo así:

        1  <!DOCTYPE html>
        2  <html>
        3    <head>
        4    </head>
        5    <body>
        6    </body>
        7  </html>

Esta sección contiene todo el contenido visible de la página web. Aquí es donde se estructura la mayor parte del contenido que los usuarios verán
