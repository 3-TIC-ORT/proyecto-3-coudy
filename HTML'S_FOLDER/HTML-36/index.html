El elemento de imagen <img> representa una imagen en el documento.

Sintaxis:

<img src="coudyIcon.png" alt="Coudy"/>

¿Para qué sirven estas partes del elemento?
  • alt = Este atributo define el texto alternativo que describe la imagen, texto que los usuarios verán si la URL de la imagen es errónea, la imagen tiene un formato no soportado o si la imagen aún no se ha descargado
  • src = La URL de la imagen
