Listas ordenadas

El elemento de listas ordenadas (<ol></ol>) permite definir listas ordenadas ("Ordered List")

Sintaxis:

<ol>

<li>Punto 1</li>

<li>Punto 2</li>

<li>Punto 3</li>

</ol>

Así se vería la línea de código anterior:

  1. Punto 1
  2. Punto 2
  3. Punto 3
