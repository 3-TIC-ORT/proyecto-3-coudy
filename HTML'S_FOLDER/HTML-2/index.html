HTML (HyperText Markup Language)

Es un lenguaje de marcado que indica a los navegadores web cómo estructurar las páginas web que visita.

Este lenguaje define la estructura y el contenido básico que los navegadores web interpretan y muestran a los usuarios. La anatomía de un documento HTML es esencial porque permite organizar y presentar la información de manera coherente y accesible.
