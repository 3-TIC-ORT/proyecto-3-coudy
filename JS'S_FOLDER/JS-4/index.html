Variables

Una variable es un lugar con un nombre específico que guarda un valor. Esto permite acceder a ese valor utilizando el nombre asignado.

En JavaScript, existen muchas maneras para declarar variables, pero vamos a ver dos principales. Estas son:

let (variable mutable):

Se usa cuando el valor de la variable puede cambiar en algún momento del programa.

Ejemplo básico:

let edad = 25; // Esto es un comentario

edad = 30; // Cambiamos el valor de la variable

Al final, la variable “edad” va a tener el valor de 30
