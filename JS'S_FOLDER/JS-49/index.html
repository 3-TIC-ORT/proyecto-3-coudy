DOM

El método appendChild() es una función utilizada para agregar un nodo hijo a otro nodo en el DOM. Es especialmente útil cuando queremos insertar elementos nuevos, como párrafos, imágenes o botones, dentro de otros elementos ya existentes en una página web.

Sintaxis:
elementoPadre.appendChild(nuevoElemento);

  • elementoPadre: Es el nodo en el cual vamos a agregar un nuevo elemento.
  • elementoHijo: Es el nodo que se insertará como hijo.
