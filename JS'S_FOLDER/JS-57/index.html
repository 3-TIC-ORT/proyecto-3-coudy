Listas

Una lista en JavaScript, conocida como array, es una estructura de datos que permite almacenar múltiples valores en una sola variable. A diferencia de las variables comunes, que solo pueden contener un valor a la vez, un array puede contener una colección de elementos, como números, cadenas de texto u otros tipos de datos.

Ejemplo de sintaxis:
let frutas = ["manzana", "banana", "naranja"];

Cada elemento en una lista tiene una posición o índice, que comienza desde el número 0. Esto significa que el primer elemento se encuentra en la posición 0, el segundo en la posición 1, y así sucesivamente.
