console.log

console.log() es una herramienta en JavaScript que nos permite ver mensajes en la consola del navegador.

La consola del navegador es una herramienta que nos permite ver y probar nuestro código JavaScript mientras estamos en una página web.

Ejemplo básico:

console.log("Hola Daro");// La consola va a imprimir el texto "Hola Daro"
