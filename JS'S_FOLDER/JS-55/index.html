Repetición

Sintaxis:
for (expresión-inicial; condición; expresión-final) {
     // Código a ejecutar en cada iteración
}

Ejemplo básico:
// Imprimir los números del 1 al 5
for (let i = 1; i <= 5; i++) {
     console.log(i);
}

  • Expresión inicial: Es el punto donde definimos o asignamos valor a la variable que llevará la cuenta del bucle.
  • Condición: Es la regla que se verifica en cada repetición del bucle.
    • Si esta condición es verdadera, el bucle sigue.
    • Si es falsa, el bucle termina.
  • Expresión final: Es lo que hacemos al final de cada vuelta del bucle. Normalmente, aquí aumentamos o cambiamos la variable que cuenta las repeticiones.
    Ejemplo: i++ (Aumenta el valor de i en 1 después de cada vuelta).
