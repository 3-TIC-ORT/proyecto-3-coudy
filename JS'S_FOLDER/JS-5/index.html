Variables

Una variable es un lugar con un nombre específico que guarda un valor. Esto permite acceder a ese valor utilizando el nombre asignado.

En JavaScript, existen muchas maneras para declarar variables, pero vamos a ver dos principales. Estas son:

const (variable constante):

Se usa cuando el valor no debe cambiar durante la ejecución del programa.

Ejemplo básico:

const edad-Ivo = 25

edad-Ivo = 50 // Esta línea causaría un error, ya que "edad-Ivo" es constante y no se puede cambiar
