Repetición

while: Crea un bucle que ejecuta una sentencia especificada mientras cierta condición se evalúe como verdadera. Este bucle se suele utilizar cuando no se conoce de antemano cuántas iteraciones se deben realizar.

Sintaxis:
while(condición){
     sentencia
}

  • (condición): Aquí se especifica una expresión que se evaluará como verdadera o falsa.
    • Si la condición es verdadera, se seguirá ejecutando el bloque de código dentro del bucle.
    • Si la condición es falsa, el bucle se detendrá y el programa continúa con la siguiente línea de código después del bucle.
  • {sentencia}: Las llaves delimitan el bloque de código que se ejecutará mientras la condición sea verdadera.
