Repetición

for: Crea un bucle que consiste en tres expresiones opcionales, encerradas en paréntesis y separadas por puntos y comas, seguidas de una sentencia ejecutada en un bucle. Este bucle se suele utilizar cuando se conoce de antemano el número de repeticiónes que se deben realizar.

Sintaxis:

for (expresión-inicial; condición; expresión-final) {
     // Código a ejecutar en cada iteración
}
