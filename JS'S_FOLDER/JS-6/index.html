Datos

  • Number: es un tipo de dato que se utiliza para representar valores numéricos, ya sean enteros o decimales.
  • String: Almacena texto. Cualquier conjunto de letras, palabras o símbolos. Se guarda entre comillas (pueden ser simples ', dobles " o acento grave `).

Ejemplos:

let edad = 16;

let decimal = 3.14;

Ejemplos:

let nombre = "Ana";

let saludo = 'Hola, ¿cómo estás?';

let plantilla = `Mi edad es 25 años.`;
