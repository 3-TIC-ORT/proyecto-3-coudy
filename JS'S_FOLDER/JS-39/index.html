DOM

El método getElementById() es utilizado para acceder a un elemento del DOM (Document Object Model) mediante su identificador único (ID). Este se suele utilizar para manipular y modificar elementos de un documento HTML
Sintaxis:
document.getElementById("id-del-elemento")

  • document: Representa la página web (el "documento" HTML) completa.
  • getElementById: Significa "obtener el elemento por su ID".
  • "id-del-elemento": Es el nombre que declaraste al ID en tu documento HTML. Este nombre siempre debe escribirse entre comillas.

Recordá que...
El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento.
