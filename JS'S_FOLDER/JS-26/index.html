Operadores

  • El operador NOT invierte el valor de una condición. Si es verdadero, lo vuelve falso, y viceversa.

Ejemplo:
let edad = Number(prompt("¿Cuál es tu edad?"));
if (!(edad < 18)) {
    alert("Acceso denegado. Debes ser mayor de edad");
} else {
    alert("Eres myor de edad ¡Bienvenido al sitio!");
}

// Al utilizar un operador NOT en este ejemplo, si ingresas un número de 18 para arriba te denegará el acceso. En cambio, si ingresas un número de 17 para abajo, te permitirá el acceso
