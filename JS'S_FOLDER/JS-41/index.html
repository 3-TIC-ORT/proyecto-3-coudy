addEventListener() es un método que permite a un elemento de tu página web reaccionar a ciertos eventos, como clics o movimientos del mouse. El objeto específico puede ser un simple elemento en un archivo, el mismo documento o una ventana. Este método permite que una función se ejecute cuando ocurre un evento específico en el elemento.

Sintaxis:
elemento.addEventListener(“evento”, función);

  • Elemento: Es el objeto del cual se detecta el evento, como un botón, una imagen o una sección del sitio web.
  • Evento: Es la acción que el navegador está esperando, como un clic ('click'), un movimiento del mouse ('mouseover'), o una tecla presionada ('keydown').
  • Función: Es el bloque de código que se ejecuta cuando ocurre un evento.
