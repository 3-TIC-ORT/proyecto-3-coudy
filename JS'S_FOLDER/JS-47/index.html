El método document.createElement() es utilizado para crear nuevos elementos HTML de manera dinámica mediante JavaScript. Con este método, puedes añadir contenido a tu página web sin necesidad de que el elemento exista desde un principio en el archivo HTML.
Sintaxis:
let nuevoElemento = document.createElement(nombreDelElemento);

  • nombreDelElemento: Es una cadena de texto que representa el tipo de elemento HTML, según se etiqueta (tag) semántica que deseas crear. EJ( 'div', 'p', 'button',).

Ejemplo:
let párrafo = document.createElement('p');
