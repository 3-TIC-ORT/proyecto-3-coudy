Listas

Es posible modificar el valor de un elemento en un array indicando su índice.
Ejemplo:
let frutas = ["manzana", "banana", "naranja"];
frutas[0] = "pera"; // Ahora el primer elemento es "pera"

El array tiene una propiedad llamada length, que indica el número de elementos que contiene en la consola.
Ejemplo:
let frutas = ["manzana", "banana", "naranja"];
console.log(frutas.length); // Va a imprimir el número 3
