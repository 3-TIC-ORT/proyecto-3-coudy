Listas

Para acceder a un elemento específico de un array, se usa el índice del elemento dentro de los corchetes [].
Ejemplo:
let frutas = ["manzana", "banana", "naranja"];
console.log(frutas[0]); // Va a mostrar: "manzana"
console.log(frutas[1]); // Va a mostrar: "banana"

Para añadir un elemento al final de un array, se utiliza el método push().
Ejemplo:
let frutas = ["manzana", "banana", "naranja"];
frutas.push("uva"); // Añade "uva" al final del array
