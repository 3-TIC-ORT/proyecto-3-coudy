Funciones

La declaración return dentro de una función detiene su ejecución y devuelve un valor al lugar donde se llamó la función. Es decir, cuando se encuentra return, la función deja de ejecutarse, y el valor especificado se envía de vuelta al llamador. Esto permite que el resultado de la función sea utilizado en otras partes del código.

Ejemplo básico:
function sumar(a, b) {
    return a + b; // Devuelve la suma de a y b
}
let resultado = sumar(5, 3);
console.log(resultado); // Imprime 8
