If… Else

El método if...else permite que el programa decida qué acción tomar según si una condición se cumple o no. Funciona como una instrucción que le dice al código: "Si pasa esto, hace una cosa; si no, hace otra".

Sintaxis:
let edad = prompt("¿Qué edad tienes?");
if (edad >= 18) { // Si edad es mayor o igual a 18
    alert("Eres mayor de edad");
} else { // Si edad es menor a 18
    alert("Eres menor de edad");
}
