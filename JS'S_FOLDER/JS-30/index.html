Funciones

Una función es un bloque de código reutilizable que se puede ejecutar cuando sea necesario. Las funciones ayudan a evitar repetir código, mejorar la organización y facilitar el mantenimiento del programa.

Sintaxis básica:
function nombreDeLaFuncion(parametro1, parametro2) {
    // Código a ejecutar
}

Partes de una función:

  • El nombre de la función.
  • Una lista de parámetros de la función, entre paréntesis y separados por comas.
  • El código a ejecutar, encerrado entre llaves, { ... }
