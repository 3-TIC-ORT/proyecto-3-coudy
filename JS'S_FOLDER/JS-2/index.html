JavaScript

Es un lenguaje de programación que se puede aplicar a un documento HTML y usarse para crear interactividad dinámica en los sitios web.

Para linkear un archivo JavaScript en uno HTML introduce el siguiente código en una nueva línea, justo antes de la etiqueta de cierre </body>:

Sintaxis:

<body>

<script src="main.js"></script>

</body>
