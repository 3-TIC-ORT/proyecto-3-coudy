Listas

Si queremos recorrer un array uno por uno, podemos utilizar un bucle for de la siguiente manera:

let frutas = ["manzana", "banana", "naranja"];
for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

Esto va a mostrar en la consola lo siguiente:
manzana
banana
naranja
