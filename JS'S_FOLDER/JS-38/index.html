DOM

El DOM (Modelo de Objetos del Documento) es un conjunto de herramientas que permite representar y manipular documentos HTML. Se carga en el navegador como un árbol de nodos, donde cada nodo corresponde a una parte del documento, como un elemento, un texto o un comentario. Algunos de las propiedades y métodos que se utilizan para la manipulación del DOM son:

  • document.getElementById
  • .addEventListener
  • .value
  • textContent
  • .createElement
  • .appendChild
