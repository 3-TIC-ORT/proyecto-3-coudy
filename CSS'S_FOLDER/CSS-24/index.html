Box Model

El modelo de caja CSS es básicamente un cuadro que envuelve cada elemento HTML. Este está formado por:

  • El margen de la caja (“margin”): El margen es la capa más externa. Envuelve el contenido, el relleno y el borde como espacio en blanco entre la caja y otros elementos.
