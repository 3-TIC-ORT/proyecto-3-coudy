Box sizing

content-box: (o modelo de cajas estándar): Utilizando esta propiedad, el ancho y alto que defines para un elemento solo se aplican al área de contenido (content). Con esta propiedad, el elemento va a medir la suma del tamaño del content, padding y border. El márgen (margin) no modifica el tamaño del elemento.

.caja {

width: 350px;

height: 150px;

margin: 10px;

padding: 25px;

border: 5px solid black;

}

En este ejemplo, el espacio que va a ocupar esta caja usando el modelo de cajas estándar será en realidad de 410 px (350 + 25 + 25 + 5 + 5); y su altura, de 210 px (150 + 25 + 25 + 5 + 5)
