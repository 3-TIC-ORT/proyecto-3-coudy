Anatomía de una regla CSS

En CSS las reglas de estilo siguen una estructura específica que facilita la personalización de la apariencia de los elementos. Los tres componentes fundamentales de una de ellas son:

p {

color: red;

}

A la derecha de la propiedad, después de los dos puntos (:), tienes el valor de la propiedad, para elegir una de las muchas posibles apariencias para una propiedad determinada
