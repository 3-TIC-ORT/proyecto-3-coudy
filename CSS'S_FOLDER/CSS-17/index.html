Propiedades - Font

  • font-size: especifica el tamaño de la letra de un texto. Acepta valores numéricos (EJ: 20px) o valores ya definidos (EJ: small, medium, large, etc)
  • font-style: permite definir el aspecto de una familia tipográfica entre los valores:
  • font-family: especifica la fuente de un texto. Ejemplo: Arial, ComicSans, etc.
  • font-weight: especifica el grosor de la letra. Esta puede ser referida en "pesos números"(EJ: 20px) o en valores ya definidos (EJ: Bold, normal, etc)
