Jerarquía

El flexbox es un sistema de diseño en CSS que permite organizar y alinear elementos en una página web de manera flexible y eficiente.

Conceptos básicos:

  • Contenedor Flex: Este es el elemento padre que tiene la propiedad display: flex. Al aplicar display: flex, todos los elementos hijos dentro de este contenedor se organizarán de manera flexible y ajustable según las reglas que establezcas.
  • Elementos Flex: Son los elementos hijos contenidos dentro del contenedor Flex. Estos elementos se ajustan y distribuyen según las propiedades definidas en el contenedor padre.
